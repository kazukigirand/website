# Art Images Directory

This directory contains artwork images for the gallery.

## Expected Structure

```
/images/art/
  ├── neural-garden.jpg
  ├── neural-garden-thumb.jpg
  ├── flow-field-3.jpg
  ├── flow-field-3-thumb.jpg
  └── ... (other artworks)
```

## Image Guidelines

- **Full images**: 1920x1080 or similar aspect ratio
- **Thumbnails**: 400x400 square crops
- **Format**: WebP or JPG for best performance
- **Optimization**: Use Next.js Image component for automatic optimization

## Placeholder Usage

Until real images are added, the gallery uses gradient placeholders 
generated via CSS. Simply add your images with the correct filenames 
to replace the placeholders.
